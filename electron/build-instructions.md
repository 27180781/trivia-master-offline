# הוראות בניית קובץ EXE למשחק טריוויה

## דרישות מקדימות

1. **Node.js** מותקן במחשב (גרסה 18 ומעלה)
2. **Git** מותקן (אופציונלי, להורדת הפרויקט)

---

## שלב 1: הכנת המשחק

### א. ייצוא חבילת המשחק
1. גש לכתובת `/package` באפליקציה
2. העלה קובץ Excel עם השאלות
3. הזן שם למשחק
4. לחץ על "הצג קוד להטמעה"
5. העתק את הקוד

### ב. הטמעת המשחק בפרויקט
1. פתח את הקובץ `src/data/embedded-game.json`
2. מחק את התוכן הקיים
3. הדבק את הקוד שהעתקת
4. שמור את הקובץ

---

## שלב 2: בניית האפליקציה

פתח טרמינל בתיקיית הפרויקט והרץ:

```bash
# התקנת תלויות (פעם ראשונה בלבד)
npm install

# בניית האפליקציה
npm run build
```

---

## שלב 3: הכנת Electron

עבור לתיקיית electron והרץ:

```bash
cd electron

# התקנת תלויות Electron (פעם ראשונה בלבד)
npm install
```

---

## שלב 4: בניית קובץ EXE

### אפשרות א: קובץ Portable (ללא התקנה)
```bash
npm run build:win
```

הקובץ יווצר בתיקייה: `electron/release/`

### אפשרות ב: קובץ התקנה (Installer)
אותה פקודה תייצר גם קובץ התקנה NSIS.

---

## שלב 5: שליחה ללקוח

1. מצא את הקובץ בתיקייה `electron/release/`
2. שלח את קובץ ה-EXE ללקוח
3. הלקוח יפעיל ויתחיל לשחק מיד

---

## פתרון בעיות

### הקובץ גדול מדי
- קובץ Electron כולל דפדפן Chromium (~150MB)
- זה נורמלי ואי אפשר להקטין משמעותית

### האפליקציה לא נפתחת
- ודא שהקובץ `dist/index.html` קיים
- הרץ `npm run build` מחדש

### שגיאת הרשאות ב-Windows
- הרץ את הטרמינל כמנהל (Administrator)

---

## טיפים

- **בדיקה לפני אריזה**: הרץ `npm run start` בתיקיית electron לבדיקה
- **שינוי אייקון**: החלף את `public/favicon.ico` באייקון מותאם
- **שינוי שם האפליקציה**: ערוך את `productName` ב-`electron/package.json`
